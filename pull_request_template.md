## Checklist de Lançamento
*Antes de fazer o merge, confirmar se concluíste os seguintes pontos:*

### Estrutura e Código
- [ ] **Refatoração:** O código está limpo e segue o padrão do projeto?
- [ ] **Nomenclatura:** Variáveis, funções e ficheiros estão alinhados com o `master`?
- [ ] **Organização:** As pastas estão nos locais corretos?

### Documentação
- [ ] **README/Wiki:** As alterações foram documentadas?
- [ ] **Comentários:** Código complexo foi explicado?

### Validação
- [ ] Testei as alterações localmente e tudo funciona.
- [ ] Removi `console.log` ou prints de debug.
