## Mocks
---
Contem variáveis publicas, compatíveis com estrutura JSON, para usos diversos no código. Deve ser alterado com frequência em razão de testes futuros. Documentar alterações sempre que possível.


`criadores.ts`: array de id de criadores.\
`guilds.`: array de servidores, contendo id, nome de exibição e ícone.\
`mensalistas.ts`: array de id de mensalistas.\
`polls.ts`: array de enquetes/rascunhos.
